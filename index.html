FAQ

Como usar os filtros de Período e GP?

Selecione os valores no topo da Home. Ao abrir qualquer dashboard, os filtros são propagados automaticamente via parâmetros de URL.

O que significa cada badge de status?

Atualizado (verde) = carregado. Atualizando... (amarelo) = em carregamento. Erro de conexão (vermelho) = falha — use o botão Retry no card ou o Atualizar global.

Como reportar um problema em um dashboard?

Clique no botão 🚨 no card correspondente ou em "Reportar Problema" no topo. O ticket vai automaticamente para Felipe Vinicius no Ekyte.

O dashboard está como "Em desenvolvimento". O que fazer?

Significa que a URL ainda não foi configurada. Fale com o Tech Lead para adicionar o link ao portal.
